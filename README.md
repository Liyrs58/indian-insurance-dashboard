# Indian Insurance Market Dashboard

Bloomberg-terminal style dashboard for Indian insurance industry data.

## Preview

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## Data Sources
- IRDAI Annual Reports
- IRDAI Handbook of Indian Insurance Statistics
- Company annual filings (LIC, ICICI Prudential, HDFC Life, etc.)

## Stack
- HTML/CSS (Bloomberg terminal theme)
- Chart.js
- JetBrains Mono font
